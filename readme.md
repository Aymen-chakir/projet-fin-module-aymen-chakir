# Delicious – Restaurant Website (HTML & CSS Final Project)

##  Description
**Delicious** is a modern, clean, and responsive restaurant website built as a Final Module Project using HTML, CSS, and SASS.  
The website includes the essential sections every restaurant site needs: **Home, Menu, About, and Contact**, all designed with a polished layout using Flexbox and CSS Grid.

---

##  Features
-  **Fully Responsive Design** using Flexbox & CSS Grid  
-  **Smooth interactions** using pseudo-classes: `:hover`, `:focus`, `:active`  
- **Custom styling** using pseudo-elements: `::before`, `::after`  
- **SASS Features**: Variables & Mixins  
- Clean and well-organized structure  
-  Balanced layout between text and images  

---

##  Technologies Used
- **HTML5**  
- **CSS3**  
- **SASS (SCSS)**  
- **Flexbox**  
- **CSS Grid**
